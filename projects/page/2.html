Artesyn Embedded Power

Develop a PHP web-based system that will record the detection of model product defects in AOI Machine.

Web Application
  • Login system for individual user.
  • Add a model product for subjected detection data with timestamp.
  • Upload individual photos for model product.
  • User can delete model product including their defect data.
  • Using AJAX to create a monthly defect data.
  • Member information view list.
  • Responsive design.
  • Deployed in Heroku cloud-platform free hosting plan.
